## Features

## Tech Stack

**Client:** React, TypeScript, Styled-Components




## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install or yarn install
```

Start the server

```bash
  npm run start or yarn start
```
